# Linux Security Log Investigation Project

## Objective
To analyze Linux authentication logs and identify malicious login attempts and privilege escalation activity.

## Tools Used
- Linux (Ubuntu)
- grep, awk, tail
- journalctl

## Key Findings
- Detected multiple failed SSH login attempts
- Identified attacker IP addresses
- Analyzed targeted usernames
- Observed failed privilege escalation attempts

## Skills Demonstrated
- Linux log analysis
- Security investigation
- Incident detection
- SOC analyst fundamentals
